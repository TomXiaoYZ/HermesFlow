## 🔄 方法: Start / Stop / Continue

### ✅ Continue（继续做）

_什么做得好？我们应该继续做什么？_

| 内容 | 投票数 | 备注 |
|------|--------|------|
| 例：每日站会高效 | ⭐⭐⭐ | 15分钟内完成 |
|  |  |  |
|  |  |  |

### 🚀 Start（开始做）

_我们应该开始做什么？_

| 内容 | 投票数 | 备注 |
|------|--------|------|
| 例：每周技术分享 | ⭐⭐ | 提升团队技能 |
|  |  |  |
|  |  |  |

### 🛑 Stop（停止做）

_什么做得不好？我们应该停止做什么？_

| 内容 | 投票数 | 备注 |
|------|--------|------|
| 例：会议太多 | ⭐⭐⭐⭐ | 每天 >3 小时 |
