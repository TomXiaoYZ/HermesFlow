# HermesFlow 量化交易平台架构文档

## 系统概述

HermesFlow是一个高性能的量化交易平台，支持多交易所、多链的数据接入和交易执行，具备完整的策略开发、回测、风控和执行功能。系统采用微服务架构，保证高可用性和可扩展性。

## 技术栈选择

### 后端技术栈
- 核心服务：Rust（高性能交易引擎）
- 业务服务：Golang（API服务）
- 数据处理：Python（数据分析和策略回测）

### 前端技术栈
- 框架：React + TypeScript
- 状态管理：Redux Toolkit
- UI组件：Ant Design Pro
- 图表库：TradingView + ECharts

### 数据存储
- 实时数据：Redis
- 历史数据：ClickHouse
- 业务数据：PostgreSQL
- 消息队列：Kafka

### 基础设施
- 容器编排：Kubernetes (EKS)
- 服务网格：Istio
- 监控：Prometheus + Grafana
- 日志：ELK Stack
- 对象存储：AWS S3
