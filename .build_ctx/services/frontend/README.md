# Frontend Service

HermesFlow 前端服务 - 使用 React 和 TypeScript 构建的现代化单页应用。

## 🚀 功能

- ✅ React 18
- ✅ TypeScript 支持
- ✅ 响应式设计

## 🛠️ 技术栈

- **框架**: React 18.2
- **语言**: TypeScript 5.3

## 📦 本地开发

```bash
npm install
npm start
```

## 🧪 测试

```bash
npm test
```

## 🐳 Docker

```bash
docker build -t frontend:latest .
docker run -p 80:80 frontend:latest
```
